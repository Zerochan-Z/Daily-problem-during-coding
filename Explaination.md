### When to use

| Situation | Use docstring ('''''')? |
|-----------|----------------|
| Quick script for yourself | No, `#` comment is fine |
| Function you will reuse | ✅ Yes |
| Library/package for others | ✅ Yes (required) |
| Team project | ✅ Yes |
| LeetCode/CodingBat problem | No (wasted time) |
